## Description

**proglass** The style has been redesigned for phpbb 3.3.x to fix the limitation of the fixed width images. 
The mod is no longer needed for phpbb3.3 and the width can be easily changed between the original size and 1028px. 
It won't look good if you go any wider than that.

## Features

- Clean design
- Responsive
- Optimized for performance
- Compatible with phpBB 3.x
